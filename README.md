## Rechtsgeschäfte

Zweiseitige Rechtsgeschäfte
- synallagmatisch               (Gegenleistungen)
- unvollkommen zweiseitig       (Eine Partei trägt Hauptlast)
- einseitig verpflichtend       (Eine Partei wird verpflichtet)

Einseitige Rechtsgeschäfte
- streng einseitig              (wirkt - ganz ohne, dass es jemand wahrnimmt)
- einseitig empfangsbedürftig   (wrkt nur nach Kenntnisnahmemöglichkeit)

## Auslegungsmethoden Willenserklärung

natürlich: Auslegung nach wirklichem Willen §133 BGB
normativ: Auslegung nach dem tatsächlich Gesagten, Verkehrssitte, Vertrauensschutz

# Vertretung §164 BGB

- Vertretung muss zulässig sein
- Vertreter muss offenkundig, im Namen handeln
- Vertreter muss eigene Willenserklärung abgeben, Ermessungsmacht haben
- Vertreter muss Vertretungsmacht haben
